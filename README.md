# PA-CODERS 😈

### GAME WHEEL OF DOOM

## Introduction 📌

The project called "PC-CODERS" is a Wheel of Doom, based on JAVASCRIPT, CSS and HTML. The goal is to create a fair and random sacrifice selection system for the coders, with the possibility of adding more coders.

## Practice goals 📑

1.- Improve project management with Scrum.

2.- Make better Sketches and Mockups (design tools like Miró and Figma.)

3.- GIT and GitHub manipulation and teamwork flow.

4.- Semantics in the HTML code.

5.- Divide CSS into atomic components.

6.- Integrate the tools and processes for programming in JS.

7.- Use MediaQueries for the Responsive Web.

## Starting 👩‍💻

These instructions will allow you to get a copy of the project running on your local computer, for development and testing purposes:

Open your terminal and run: `git clone https://github.com/SolTuripe/PC-CODERS.git` \
To download the project, make a fork.

[To see the page in operation you can access here](https://crada2.github.io/PC-CODERS/)
